# SeeTrend


## 개발 배경
최근 여러 포털 사이트의 인기 급상승 검색어 조회가 불가함에 따라 실시간으로 화제가 되고 있는 주제에 대해 파악이 어려워졌다.
따라서 해당 사이트를 통해 인기 검색어 조회 및 검색 추이를 제공함으로써, 많은 사람들이 관심을 가지는 주제에 대해 파악할 수 있도록 개발하였다.

## 사용 라이브러리
- Selenium
- 네이버 데이터랩 API
